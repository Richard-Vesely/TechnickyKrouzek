Instructions for ChatGPT (AI assistant)

When writing Arduino code that uses an IR remote, look for which option is specified.

  • Never hardcode raw hex IR values directly in sketches.
  • For logic, always compare against constants:
    if (signal == ONE) { ... }
    switch (signal) { case LEFTBTN: ... }
  • If NEC repeat (0x0) occurs, either ignore it or reuse the last non-repeat button as appropriate.
  • When troubleshooting, always include Serial output showing both the raw code and the readable label using the constants.
  • When handling IR input, read the raw code with:
    unsigned long signal = IrReceiver.decodedIRData.decodedRawData;

Option A: Using custom library

Assume a header-only library MyIRcodes.h is installed in Documents/Arduino/libraries/MyIRcodes/.

#include <MyIRcodes.h>
using namespace MyIR;
